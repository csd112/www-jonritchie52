Glossary

Attributes
Information that can be given to elements outside of what's visible to the user..
Nested Elements
Elements with closing tags that are able to be used inside of other elements. Requires the closing tag to be wrapped inside of any parent elements.
Empty Elements
Elements with no closing tag or content.
Absolute URLs
A complete URL that includes the protocol and domain name.
Domain-relative URLs
A URl that will begin with a slash and relates to the site's domain.
